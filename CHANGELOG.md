## [1.0.0] - 2024-11-09

### Added
- Initial release of MCPsharp - Model Context Protocol server for C# project analysis
- Advanced code search with semantic understanding
- Bulk edit and refactoring capabilities
- Streaming support for large code analysis operations
- Roslyn analyzer integration with code fix providers
- Complete set of MCP tools for project analysis and manipulation
- Single-file deployment support with embedded PDB files
- Automated code fix infrastructure
- File search and symbol navigation
- Text search capabilities
- Extract method refactoring tool
- Rename symbol refactoring tool
- Support for multi-project analysis

### Fixed
- Resolved all compilation errors and warnings
- Fixed build configuration for Windows and cross-platform deployment
- Resolved PDB embedding for single-file executable deployment
- Fixed analyzer autoload configuration
